
# Materiais

Os materiais utilizados no projeto foram:
- 2x ESP32.
- 2x Protoboard's.
- Módulo Sensor de Distância Ultrassônico (HC-SR04).
- Kit Chassi 2WD (2 rodas) Robô para Arduino.
- Botão 12x12 Vermelho
- Display de 7 segmentos (4 digitos).
- Módulo RTC.
